Welcome to mhxedit's Pages.

Description

mhxedit is a Savegame Editor for the game Monster Hunter X.

It can edit some character values and your Item Box IDs and Counts. You can also search for a specific item (by its english translated name) and add that item to your item box.

Requirements

MHX Savegame exported from your 3ds sdcard. Look elsewhere about how to do this.

Usage

Load the save from the file menu and select the character slot which you want to edit from the drop down. Check the tabs, the rest is pretty straight forward and don't foget to hit save before you close the program.

I tested this with some binary compares, so it shouldn't screw up your savegame, confirmed it's working with my own saves.
